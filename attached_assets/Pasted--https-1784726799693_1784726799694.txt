2. תיקון החץ בפוטר

החץ בפוטר חייב להחזיר את המשתמש לתחילת העמוד גם במובייל וגם בדסקטופ.

יש לוודא שבתחילת העמוד קיים יעד תקין:

<body id="top">

או לחלופין:

<div id="top"></div>

החץ בפוטר צריך להיות קישור אמיתי:

<a class="footer-top-link" href="#top" aria-label="חזרה לראש העמוד">
  <span aria-hidden="true">↑</span>
</a>

בנוסף, יש להוסיף גלילה חלקה:

html {
  scroll-behavior: smooth;
}

אם בגלל JavaScript, תפריט קבוע או מבנה העמוד הקישור #top אינו מגיע ממש לתחילת העמוד, יש להוסיף גם טיפול מפורש:

document.querySelector('.footer-top-link')?.addEventListener('click', (event) => {
  event.preventDefault();

  window.scrollTo({
    top: 0,
    behavior: 'smooth'
  });

  history.replaceState(null, '', '#top');
});

יש לוודא:

שהחץ עובד בלחיצה אחת.
שהעמוד מגיע ממש לחלק העליון.
שהפעולה עובדת ב־Safari באייפון, Chrome במובייל ובדסקטופ.
שהחץ נשאר קטן ועדין.
שאין טקסט „חזרה לראש העמוד”.
שאין שינוי בשאר הפוטר.
