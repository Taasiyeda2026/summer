# תיקונים מרוכזים לקודקס: עמוד ראשון, כרטיסים, מכתב מקדים ותווית כיתות בכותרת

## רקע

התיקונים מתייחסים לקטלוג הסדנאות ב־`workshops.html`, בעיקר לעמוד הראשון כאשר מסומן “מכתב מקדים”, ובנוסף לשורת הכותרת בכל עמודי הסדנאות.

אין לשנות את מבנה העמודים הרגילים מעבר למה שמוגדר כאן.
אין לשנות את עמודי חדרי הבריחה/חטיבה/מנגנוני הבחירה.
אין לשנות את הלוגיקה הכללית של הקטלוג הרציף לפי תחומים.

---

# 1. עמוד ראשון בלבד — כרטיסי 3×2 חייבים להציג את משפט התיאור

## הבעיה

בעמוד הראשון, בחצי התחתון, יש גריד של 6 כרטיסים במבנה 3×2.
כרגע הכרטיסים אינם מתנהגים כמו הכרטיסים בעמודים הבאים, כי משפט התיאור הקצר `shortLine` נעלם.

## דרישה מחייבת

כל 6 הכרטיסים בעמוד הראשון חייבים לכלול:

* גלולת כותרת עם שם הסדנה
* תמונה
* משפט תיאור קצר מתוך `shortLine`
* תוויות תחתונות
* זום תמונה בלחיצה

אסור להסתיר את `shortLine` בעמוד הראשון.

אם אין מספיק מקום:

* להקטין מעט את התמונה
* להקטין מעט ריווחים פנימיים
* להקטין מעט את גובה הכותרת/התוויות
* אבל לא למחוק ולא להסתיר את משפט התיאור

יש לוודא שאין CSS שמסתיר תיאור בעמוד הראשון, למשל:

```css
.cover-bottom-grid .short-line {
  display: none;
}
```

אם קיימת פונקציה נפרדת שמרנדרת כרטיסים לעמוד הראשון — לבטל אותה או להתאים אותה כך שתשתמש באותו מבנה כרטיס של העמודים הרגילים.

---

# 2. עמוד ראשון בלבד — שיפור החצי העליון של המכתב המקדים

## הבעיה

בחצי העליון של העמוד הראשון, בלוק המכתב נראה קטן מדי, צפוף מדי למעלה, ומשאיר שטח לבן גדול.

## דרישה מחייבת

יש לעצב את `cover-letter-block` כך שהתוכן יתפרס יפה בתוך כל החצי העליון.

המטרה:

* להגדיל מעט את הכותרות והטקסט
* לפזר את התוכן לגובה בצורה מאוזנת
* לצמצם שטח לבן מיותר
* לשמור על כל הטקסט בתוך החצי העליון
* לא למחוק פסקאות או תוכן מהמכתב

כיוון CSS אפשרי:

```css
.cover-letter-block {
  flex: 0 0 50%;
  max-height: 50%;
  overflow: hidden;
  display: flex;
  flex-direction: column;
  justify-content: space-between;
  padding: 8mm 10mm;
}
```

אפשר להגדיל מעט טקסטים, לדוגמה:

```css
.cover-letter-block h1,
.cover-letter-block h2 {
  font-size: 22px;
  line-height: 1.25;
  margin-bottom: 3mm;
}

.cover-letter-block p,
.cover-letter-block li {
  font-size: 13px;
  line-height: 1.45;
}
```

אם יש גלישה, מותר להדק ריווחים עד 10% בלבד.
אסור למחוק טקסט מהמכתב.

---

# 3. העברת תווית הכיתות לשורת הכותרת

## הבעיה

התווית `כיתות א׳–ג׳` מופיעה כיום בתוך הכרטיסים או באזור התוויות.

## דרישה מחייבת

יש להעביר את תווית הכיתות לשורת הכותרת של העמוד.

הכותרת צריכה להיות:

```text
סדנאות STEM חווייתיות | כיתות א׳–ג׳
```

או בעיצוב שקול:

* כותרת: `סדנאות STEM חווייתיות`
* תגית צמודה באותה שורה: `כיתות א׳–ג׳`

התווית צריכה להופיע בשורת הכותרת:

* בעמוד הראשון עם המכתב המקדים
* בכל עמודי הסדנאות הרגילים
* במסך
* בהדפסה

---

# 4. שינוי תוויות בתוך הכרטיסים

לאחר שתווית הכיתות עוברת לכותרת העמוד, אין להציג בתוך כל כרטיס את התווית:
`כיתות א׳–ג׳`

בתוך הכרטיסים יש להשאיר רק:

* תוצר
* תחום
* מחיר

כלומר שורת התוויות בכרטיס צריכה להיות קצרה ונקייה יותר.

אסור שתהיה כפילות של `כיתות א׳–ג׳` גם בכותרת וגם בכרטיסים.

---

# 5. שמירה על מבנה העמוד הראשון

כאשר מסומן “מכתב מקדים”, עמוד ראשון חייב להישאר במבנה הבא:

1. Header עליון עם לוגו — ללא שינוי.
2. חצי עליון: מכתב מקדים.
3. שורת כותרת: `סדנאות STEM חווייתיות | כיתות א׳–ג׳`.
4. חצי תחתון: 6 כרטיסים בדיוק בגריד 3×2.
5. Footer ושורת קשר — ללא שינוי.

אסור שהמכתב יגלוש לתוך אזור הכרטיסים.
אסור שהכרטיסים ייחתכו.
אסור שהעמוד יגלוש לעמוד נוסף.

---

# 6. בדיקות קבלה

## עמוד ראשון עם מכתב מקדים

1. החצי העליון מציג את המכתב בצורה מרווחת, גדולה וברורה יותר.
2. אין שטח לבן חריג בתחתית בלוק המכתב.
3. אין מחיקת פסקאות מהמכתב.
4. החצי התחתון מציג 6 כרטיסים בגריד 3×2.
5. בכל אחד מ־6 הכרטיסים מופיע `shortLine`.
6. הכרטיסים בעמוד הראשון זהים במבנה לכרטיסים בעמודים הבאים.
7. העמוד הראשון נשאר A4 אחד מדויק במסך ובהדפסה.

## כותרת ותוויות

1. בראש כל עמוד סדנאות מופיעה הכותרת:
   `סדנאות STEM חווייתיות | כיתות א׳–ג׳`
2. תווית הכיתות לא מופיעה בתוך הכרטיסים.
3. בתוך הכרטיסים מופיעות רק התוויות:
   תוצר, תחום, מחיר.
4. אין כפילות של כיתות בכותרת ובכרטיסים.

## הדפסה

1. אין חיתוך של כרטיסים.
2. אין גלישת טקסט מהמכתב.
3. הכותרת עם הכיתות מופיעה תקין.
4. העמודים הבאים לא משתנים מעבר להסרת תווית הכיתות מתוך הכרטיסים.
