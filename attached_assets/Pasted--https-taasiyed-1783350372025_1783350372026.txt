הבעיה המרכזית ב-workshops.html:
הפונקציה loadCatalogPrograms טוענת את כל catalog-data.json ומחזירה גם תוכניות לא פעילות.

צריך להוסיף פונקציה מרכזית:

```js
function isActiveProgram(program = {}) {
  if (program.isActive === false) return false;
  if (program.active === false) return false;
  if (program.is_active === false) return false;
  if (String(program.activityNo || '').includes('old')) return false;
  if (String(program.gefen || '').includes('old')) return false;
  if (String(program.title || '').includes('לא פעיל')) return false;
  return true;
}
```

ואז לעדכן את loadCatalogPrograms כך שיחזיר רק תוכניות פעילות:

```js
async function loadCatalogPrograms() {
  const response = await fetch("catalog-data.json", { cache: "no-store" });
  if (!response.ok) throw new Error(`Failed to load catalog-data.json (${response.status})`);
  const data = await response.json();
  if (!Array.isArray(data?.programs)) throw new Error("catalog-data.json is missing programs array");

  return data.programs
    .map(program => normalizeProgram(program))
    .filter(program => program.title)
    .filter(isActiveProgram);
}
```

בנוסף, חובה לוודא שכל הפונקציות שמציגות או בוחרות תוכניות משתמשות רק בתוכניות פעילות:

industry-horizon

או קוד:
