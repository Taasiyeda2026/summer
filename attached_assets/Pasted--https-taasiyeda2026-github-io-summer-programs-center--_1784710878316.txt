אין לשנות עמודים אחרים ואין לשנות את תוכן הטופס או את פעולתו.

1. הקטנת תיבת יצירת הקשר

יש להקטין משמעותית את כל תיבת הטופס:

לצמצם את הרוחב המרבי של התיבה.
לצמצם את ה־padding החיצוני.
לצמצם את הרווחים בין השדות.
להקטין מעט את גובה שדות הקלט והרשימה הנפתחת.
להקטין את גובה תיבת ההודעה.
להקטין את גודל כותרות השדות.
לשמור על שתי עמודות בדסקטופ ועל עמודה אחת במובייל.
אין להקטין את הטקסט בתוך שדות הקלט לרמה שאינה קריאה.
אין ליצור כפתור שליחה רחב. הכפתור צריך להיות קומפקטי וממורכז.

גודל מומלץ לתיבת הטופס בדסקטופ:

}

}

.form-grid select {
  height: 38px;
}

.form-grid textarea {
  min-height: 72px;
}

  font-size: .76rem;
}
2. הקטנת הטקסט שמתחת לכפתור השליחה

הטקסט:

„אם יש כיתה, בית ספר או יישוב מסוים שתרצו לקדם, ניתן לציין זאת כאן.”

צריך להיות קטן ועדין משמעותית.

להגדיר:

.privacy-note,
.contact-form .form-note {
  margin-top: 8px;
  font-size: .64rem;
  line-height: 1.35;
  color: #8997a2;
  text-align: center;
}

יש לוודא שהטקסט אינו מקבל גודל מהכללים הכלליים של הטופס.

3. הקטנת הפוטר

יש להפוך את הפוטר לשורה דקה, נקייה ומינימלית.

להסיר:
להסיר לחלוטין את הלוגו מהפוטר.
להסיר את הטקסט „חזרה לראש העמוד”.
טקסט העמותה:

הטקסט:

„תעשיידע – תעשייה למען חינוך מתקדם”

צריך להיות קטן מאוד, עדין ובשורה אחת.

.site-footer {
  padding: 10px 0;
}

.footer-inner {
  min-height: 28px;
}

.footer-inner p {
  margin: 0;
  font-size: .62rem;
  line-height: 1.2;
  color: #82909b;
  white-space: nowrap;
}

.footer-inner img {
  display: none;
}
4. החלפת „חזרה לראש העמוד” בחץ קטן

במקום הטקסט יש להציג כפתור קטן ועדין עם חץ כלפי מעלה בלבד.

הכפתור צריך:

לקשר ל־#top
לכלול aria-label="חזרה לראש העמוד"
להיות קטן ולא בולט
ללא טקסט
ללא כפתור רחב
ללא רקע כהה

מבנה מומלץ:

<a class="footer-top-link" href="#top" aria-label="חזרה לראש העמוד">
  <span aria-hidden="true">↑</span>
</a>

.footer-top-link {

5. פריסת הפוטר

בדסקטופ:

הטקסט והחץ יוצגו באותה שורה.
הפוטר יהיה ממורכז או עם ריווח עדין ביניהם.
אין להשאיר שטח ריק גדול.

במובייל:

לשמור על שורה אחת ככל שניתן.
אם אין מספיק מקום, הטקסט והחץ יכולים להישאר בשורה עם רווח קטן.
אין להחזיר את הלוגו.
אין להציג שוב את המילים „חזרה לראש העמוד”.
כללים מחייבים
אין להשתמש בריווח אותיות שלילי.
אין לשנות את פעולת הטופס.
אין לשנות כתובות מייל, שדות מוסתרים או שליחת הנתונים.
אין ליצור גלילה אופקית.
אין לשנות את שאר עיצוב העמוד.
יש לבדוק בדסקטופ ובמובייל שהפוטר ותיבת הטופס נשארים ממורכזים וקומפקטיים.
לאחר השינוי יש לציין אילו קבצים שונו.
