2. צמצום גובה ההדר

להוסיף גם:

.site-header .header-inner {
  min-height: 58px !important;
}

.site-header .brand img {
  height: 34px !important;
}

אין להקטין את הלוגו מעבר ל־34px.

3. תצוגת מובייל

בתוך:

@media (max-width: 760px)

יש להוסיף:

.site-header .main-nav {
  align-items: center !important;
  gap: 7px !important;
  padding: 12px 16px 14px !important;
}

במובייל הכפתורים יהיו אחד מתחת לשני, צרים וממורכזים — לא ברוחב כל המסך.

4. רענון גרסת CSS

בקובץ:

programs-center/index.html

לעדכן רק את גרסת הקובץ:

<link rel="stylesheet" href="./compact-controls.css?v=12">
חשוב
לא לשנות את טקסט הכפתורים.
לא להוסיף כפתור חדש.
לא לשנות את פעולת הקישורים.
לא לשנות את תוכן העמוד.
לא להשתמש בריווח אותיות שלילי.
שלושת כפתורי ההדר חייבים להיות קטנים, צרים וזהים בגודל.
